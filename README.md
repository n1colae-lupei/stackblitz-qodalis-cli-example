# Qodalis CLI — Angular Example

Minimal Angular example using [`@qodalis/angular-cli`](https://www.npmjs.com/package/@qodalis/angular-cli).

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/qodalis-solutions/stackblitz-qodalis-cli-example)

## Quick Start

```bash
npm install
npm start
```

## What's Included

- `@qodalis/angular-cli` — Angular wrapper (terminal + panel components)
- `@qodalis/cli-guid` — GUID generator plugin
- `@qodalis/cli-files` — File browser plugin
- `@qodalis/cli-users` — User management plugin
